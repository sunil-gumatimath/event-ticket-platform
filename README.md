# Tickets

Event ticket platform for booking and managing events.

## Tech Stack

- Java 21
- Spring Boot 3.2.4
- Spring Security with OAuth2
- Spring Data JPA
- H2 (dev) / PostgreSQL (prod)
- Maven

## Build

```bash
mvn clean install
```

## Run

```bash
mvn spring-boot:run
```
